# Eppyphany
A CLI Engine that procedurally generates`osu!mania` beatmaps directly from audio files.

Named as a nod to the roots of the ecosystem (`github.com/ppy`), **Eppyphany** acts as an active-feedback mapping pipeline. It analyzes the audio, drafts patterns, and benchmarks them against a port of the `osu!lazer` `ManiaDifficultyCalculator` to hit proposed star ratings.
